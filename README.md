# Google Drive API Web Application

## Project Overview
This project is a web-based application that demonstrates the integration of the **Google Drive API** using **OAuth 2.0 authentication**. It allows users to securely sign in with their Google account and perform basic file management operations such as viewing files, uploading files, and creating folders in Google Drive.

The project highlights the proper use of APIs, authentication, and client-side JavaScript integration with Google services.

---

## Features
- Google Sign-In using OAuth 2.0
- List files from Google Drive
- Upload files to Google Drive
- Create folders in Google Drive
- Secure logout and token revocation
- Black and yellow themed user interface

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Drive API v3
- Google Identity Services (OAuth 2.0)

---

## Google Cloud Configuration
- Google Cloud Project created
- Google Drive API enabled
- OAuth 2.0 Client ID (Web Application)
- Authorized JavaScript origins set to localhost
- OAuth Consent Screen configured in testing mode
- Test users added for development access

---

## How to Run the Project
1. Clone or download the project files
2. Create a Google Cloud project
3. Enable Google Drive API
4. Create an OAuth 2.0 Client ID (Web Application)
5. Add `http://localhost` to Authorized JavaScript origins
6. Insert your **Client ID** and **API Key** in `script.js`
7. Run the project using Live Server or a local web server
8. Open the app in your browser and sign in with Google
